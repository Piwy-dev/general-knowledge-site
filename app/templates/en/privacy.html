This privacy policy applies to the Discord General Knowledge application and to the website you are currently browsing.

1. What we collect

The General Knowledge application collects two types of data. On the one hand, data related to and secondly data linked to the configuration of the application on the server.

This site collects functional cookies in order to ensure its operation and an optimal user experience. experience. These include language preferences and acceptance of cookies.

1.1. User-related data

The only user-related data collected is the user ID. This identifier is collected when the user uses one of the test or profile commands. This identifier is used to link the user to his number of points to save his progress and so that he can have them later.

1.2 Configuration-related data

Configuration data includes all the data that the administrators of a server can save to keep their preferences for that server, so that they don't have to save them every time. For the moment, this only includes the language used by the application.

1.3. Other data

To ensure that the data collected remains linked to the correct server, the server identifier is also collected for each of the operations mentioned above.

2. Data storage

All data is stored in a database. Only the application administrators have access to it. Under no circumstances will this data be passed on to other entities or persons (unless requested for valid legal reasons). All administrators are named in this document (see point 5.1).

2.1. Access method

All server members can view their points or those of other members via the profile command or the general server ranking. However, they cannot change them, except in specific cases (see point 3).

3. Data modification

Data can be modified in three different ways.

3.1. By the application

A user's points are automatically modified each time they use a test command. They receive points if they answer correctly, and lose points if they don't.

3.2. By server administrators

Administrators of a server on which the application is installed can change the language used by the application on the server. They can also change a member's points. In both cases, the changes are saved automatically.

3.3. By the administrators of the application

The administrators of the application can modify the data directly in its database. However, this is never done without the agreement of the user(s) concerned.

4. Data deletion

Data is collected for an indefinite period. In principle, it is never deleted. However, it is possible to submit a request for data deletion via this link. This request will be processed as quickly as possible. As for cookies, you can delete them yourself. To do this, please refer to the method used by your browser.

5. Contact us

For any further information, you can contact an administrator.
