🇫🇷 Conditions d’utilisation

🕖 Dernière modification: le mercredi 02 mars 2022.

⚠️ Merci de lire ces conditions d’utilisation attentivement avent d’utiliser l’application: General Knowledge.

1. Informations générale

Ce bot utilise une connexion à une base de données qui sauvegarde les différentes données des utilisateurs, comme leurs points. Chaque fois que vous utilisez une commande de test vous gagnez des points si vous répondez correctement et vous en perdez dans le cas contraire.

1.1. Politique de confidentialité

Le nombre de points d’un utilisateur est enregistré, dans une base de données, à chaque utilisation d’une commande de test par celui-ci. Afin de pouvoir lier les points au bon utilisateurs, son identifiant est enregistré en même temps. L’identifiant des utilisateurs est la seule donnée que cette application collecte. Une autre donnée collectée par l’application est la langue choisie pour le serveur. Les administrateurs ont la possibilité de choisir une langue pour l’application. Celle-ci est ensuite stockée dans la base de données.

Pour plus d'informations, tu peux consulter notre politique de confidentialité.

2. Commandes de test

Pour avoir une question de vrai ou faux, utilises la commande /truefalse, ensuite réagis avec ✅ si c'est vrai, ou avec ❌ si c'est faux.

2.2. Test des drapeaux

Utilises la commande /flag pour tester tes connaissances en drapeaux. Le premier message que tu envoies par après sera considéré comme ta réponse.

2.3. Test des capitales

Utilises la commande /capital pour tester tes connaissances en capitales. Le premier message que tu envoies par après sera considéré comme ta réponse.

2.4. Test des logos

Utilises la commande /logo pour tester tes connaissances en logos. Le premier message que tu envoies par après sera considéré comme ta réponse.

3. Commandes des points

Ces commandes permettent de voir toutes les points de chaque utilisateur. Elles sont liées en permanence à la base de données. Seuls les administrateurs des serveurs ont la possibilité de modifier les points d’un utilisateur.

3.1. Profile

La commande /profile te montre tes points. Tu peux taguer quelqu'un après la commande pour voir ses points.

3.2. Classement

Pour voir le classement de serveur utilises la commande /leaderboard. Cette commande ne montre que les 20 premiers du serveur.

3.3. Changer les points

Si tu veux changer les points d'un membre, utilise la commande /adminxp. Cette commande est uniquement disponible pour les administrateurs.

4. Commandes de configuration

Les commandes de configuration permettent de configurer les paramètres du bot. Elles ne sont disponibles que pour les administrateurs.

4.1. Configuration de la langue

Configurez la langue du bot avec la commande /setlang.
