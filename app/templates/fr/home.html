General Knowledge Bot est un bot Discord de culture générale. Il propose plusieurs commandes et fonctionalité pour tester tes connaissances en culture génrale avec les membres de ton serveur. Chaque fois qu'un membre utilise une commande de test, il gagne des points si il répond correctement et perd des points dans le cas contraire. Ces points et autres options sont sauvegardés dans une base de données. Pour plus d'informations, veuillez consulter nos conditions d'utilisation et politique de confidentialité.

Multilingue

Ce bot supporte 3 langues : Français, Anglais et Néerlandais : voir comment configurer

Liste des commandes

Commandes de test

  1. /vraifaux envoie un message contenant une proposition. Une fois fait, vous pouvez cliquer sur le bouton approprié en fonction de si vous pensez que la proposition est vraie ou fausse.
  2. /drapeau envoie un message contenant un drapeau. Une fois fait, vous pouvez entrer le nom du pays qui correspond au drapeau en cliquant sur le bouton en bas du message.
  3. /capitale envoie un message contenant le nom d'un pays. Une fois fait, vous pouvez entrer le nom de la capitale de ce pays en cliquant sur le bouton en bas du message.
  4. /logo envoie un message contenant un logo. Une fois fait, vous pouvez entrer le nom de la société qui a ce logo en cliquant sur le bouton en bas du message.

Commandes des verbes irréguliers

  1. /liste-verbes envoie un message contenant la liste de tous les verbes en fonction des options suivantes :
    1. language : la langue de la liste des verbes (Anglais ou Néerlandais).
    2. showtrad : si Vrai, affiche l'infinitif du verbe et sa traduction en français ; sinon, affiche l'infinitif, l'imparfait et le participe passé.
  2. /etudier-infinitif documentation à venir ...
  3. /etudier-imparfait documentation à venir ...
  4. /etudier-participe documentation à venir ...

Commandes des points

  1. /profil affiche les points de l'utilisateur sélectionné.
  2. /classement affiche les 20 premiers membres du serveur ayant le plus de points.
  3. /adminxp modifie les points de l'utilisateur sélectionné. Cette commande est uniquement disponible pour les administrateurs.

Commandes de configuration

  1. /setlang change la langue du bot vers la langue sélectionnée (Français, Anglais, Néerlandais).
