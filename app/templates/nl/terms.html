1. General informations

This bot uses a connection to a database to save different data from the users, like their points. Each time you use a test command, you earn points if you answer correctly and lose it in the opposite case.

1.1. Privacy policy

The number of points of a user is recorded in a database each time the user uses a test command. In order to be able to link the points to the correct users, its identifier is recorded at the same time. The user identifier is the only data this application collects. Another data collected by the application is the language chosen for the server. Administrators have the option to choose a language for the application, which is then stored in the database.

For more information, you can consult our privacy policy.

3. Points commands

With these commands you can see all things related to points. They are permanently linked to the database. Only the administrators of the servers have the possibility to modify the points of a user.

3.1. Profile

The /profile command shows the points you have. You can tag someone after the command to see his points.

3.2. Leaderboard

To see the server's leaderboard use the /leaderboard command. This command only show the 20 first of the server.

3.3. Change the points

If you want to change the points of a member, use the command /adminxp. This command is only available for administrators.
